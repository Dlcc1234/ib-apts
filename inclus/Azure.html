Conseils génériques concernant Azure

Indiquer que, pour fluidifier l'expérience, les manipulation reposant purement sur le portail web n'ont pas besoin d'être réalisées à distance. Pendant la Crise des Pass Azure (plus de Pass Azure Disponibles), il a fallu se retourner systèmatiquement vers des labs hébergés (goDeploy) pour toutes les formations Azure. Il est cependant à noter que, depuis que cette situation est terminée, l'immense majorité des stages Azure ne nécessite rien d'autre que la Pass Azure fourni directement avec le stage. Le lien suivant pourra être fourni aux stagaiaires en début de formation pour mettre en place l'environnement de labs sur leur compte Azure Pass : Mise en place d'un environnement d'ateliers ib sur un compte Azure Vidéo de démonstration de la mise en place d'une connexion vNet to vNet Pour fluidifier le déroulement du stage, ne pas hésiter à conseiller aux stagiaires de réaliser l'(éventuel) exercice 0 (ou autre exercice de préparation) de chaque atelier au lancement de la présentation théorique.
